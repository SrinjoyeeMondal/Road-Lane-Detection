
# 🛣️ Road Lane Detection – Streamlit App

A computer vision web app to detect lane lines in road images using Python and OpenCV. This tool is designed to support autonomous navigation systems by highlighting lane boundaries in real-time.

---

## 🔧 Technologies Used
- Python
- Streamlit
- OpenCV
- NumPy

---

## 🚀 Features
- Upload an image and see lane detection applied instantly
- Grayscale conversion and Gaussian blur
- Canny edge detection and Region of Interest (ROI) masking
- Hough Line Transform to draw lane lines
- Real-time rendering in a clean Streamlit interface

---

## 📦 How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
